# gensample

Sample data generator.

## Building

The provided magefile can be used to build the project.

```shell
mage build
```

## Usage

1. Create a config for generating sample data. See `examples/` for example configurations.
2. Run the tool

```shell
build/gensample -c examples/demo.yml
```
